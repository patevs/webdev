# webpack-qs [![Build Status](https://travis-ci.com/patevs/webpack-qs.svg?branch=master)](https://travis-ci.com/patevs/webpack-qs)

> Webpack Quickstart Demo


## Install

```
$ npm install webpack-qs
```


## Usage

```js
const webpackQs = require('webpack-qs');

webpackQs('unicorns');
//=> 'unicorns & rainbows'
```
