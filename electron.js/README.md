# ElectronJS

> Build cross platform desktop apps with JavaScript, HTML, and CSS

---

## Installation

```bash
$ npm i -D electron@latest
# Electron   5.0.2
# Node       12.0.0
# Chromium   73.0.3683.121
```

A basic Electron app would have the following folder structure:

```text
your-app/
├── package.json
├── main.js
└── index.html
```
