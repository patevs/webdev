About

The Random Menu Selector is an interactive site that translates Robbie Porteous's physical object into a digital interactive interface using forms of coding such as HTML, CSS and Js. Designed to be used in cafe's, bars and food stalls the Random Menu Selector allows patrons to make their food choice based on their preferences well giving them an opportunity to try something new from the vendors desired specials for the day. Designed as a way to bridge the selective process between patrons and venders the Random Menu Selector for fills the needs of DSDN 112 Project 3
